# Star Wars App

Это приложение на React позволяет пользователям авторизоваться и просматривать персонажей из Star Wars с использованием данных SWAPI. Пользователь также может редактировать характеристики персонажей.

## Функционал

- Авторизация пользователей
- Просмотр персонажей из Star Wars
- Просмотр дополнительной информации, такой как homeworld, species, films и starships
- Редактирование характеристик персонажей

## Стек технологий

- **React**: Фреймворк для создания пользовательского интерфейса
- **Redux**: Управление состоянием приложения
- **React Router**: Маршрутизация между страницами
- **SWAPI (Star Wars API)**: Внешний API для данных Star Wars
- **Axios**: Для выполнения HTTP-запросов к API
